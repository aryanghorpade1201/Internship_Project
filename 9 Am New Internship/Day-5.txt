Bootstrap 
-----------
  Bootstrap  is HTML , JavaScript , CSS Framework.

  Bootstrap  is used to create  responsive website/application .

  Responsive website/application  support  all devices like smartphone, Tab, Laptop, Desktop.


   Website  :   show Product Information :>>>> HTML , CSS, JS, Bootstrap-  ReactJS

    Web Application :>>> HTML , CSS, JS, Bootstrap--mysql ---> Java

   
Container in Bootstrap
--------------------------
Container   is area  on web page where we can put elements . There two types of Container

  1)  container
  2)  container-fluid

----------

Buttons Effect
-----------------
  Bootstrap  provides us  different button to design our application.

   1)  btn
   2)  btn-primary
   3)  btn-danger
   4)  btn-secondary
   5)  btn-info
   6)  btn-warning
   7)  btn-link
   8)  btn-success


Example:
---------
<html>
<head>
	<title>Container  with Buttons </title>
	<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap@4.6.2/dist/css/bootstrap.min.css">
  <script src="https://cdn.jsdelivr.net/npm/jquery@3.7.1/dist/jquery.slim.min.js"></script>
  <script src="https://cdn.jsdelivr.net/npm/popper.js@1.16.1/dist/umd/popper.min.js"></script>
  <script src="https://cdn.jsdelivr.net/npm/bootstrap@4.6.2/dist/js/bootstrap.bundle.min.js"></script>
</head>
<body>

	<div class="container">
		  <h2>Buttons Example</h2>
           
          <button class="btn">Add Product</button>
          <button class="btn btn-primary">Add Product</button>
          <button class="btn btn-info">Add Product</button>

          <button class="btn btn-warning">Add Product</button>
          <button class="btn btn-success">Add Product</button>
          
          <button class="btn btn-secondary">Add Product</button>
          <button class="btn btn-link">Add Product</button>
          <button class="btn btn-danger">Add Product</button>
	</div>
       
     
</body>
</html>
